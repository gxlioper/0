### #MyMVC

网络IO框架:  Netty

URL映射方案: 简化版,后续考虑JRegex及参考Tomcat的映射规则

页面渲染： 0.1版只支持JSON,目前正在支持Velocity模板,后续会考虑支持freemarker,jsp.


希望借助于Netty强大的NIO能力来打造一款轻量级、性能高的HTTP服务器。

同时，并不会屏蔽技术原理，使得用户知其然且知其所以然。
