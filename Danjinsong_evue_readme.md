# evue
     

#### 介绍
二次聚合封装element-ui的form、table组件、实现数据驱动视图。http://www.evue.top/

## 文档

要查看[Live Examples](http://www.evue.top/#/form/installation/)和文档，请访问[http://www.evue.top/](http://www.evue.top/).

#### 安装
```shell
npm install evue -S
```

## 开始使用
``` javascript
 import Vue from 'vue';
 import EVue from 'evue';
 import App from './App.vue';
 
 Vue.use(EVue);
 
 Vue.use(EVue);
     new Vue({
     el: '#app',
     render: h => h(App)
 });
```
