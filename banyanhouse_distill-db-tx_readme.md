#### 介绍
测试使用distill-infra的db.Tx方法来进行事务操作

#### 软件架构
软件架构说明
distill-infra框架中集成来xormplus持久层框架，并进行了简单的封装，本项目主要用于测试主副表写入数据库的测试用例。

测试SQL文件主要位于doc目录中的init.sql文件中


#### 安装教程

测试配置文件可参考项目的config/config.toml文件
