# Future QQ.NET

为了体验一波微软爸爸的新产品，花了两小时学了下语法，写出了这玩意。

逆向出QQ协议最原本的实现方式，懂相关技术栈的都懂。

能够把你的QQ跑在服务器上，至于集群啥的不用说了。

QQ协议实现参考我的另一个项目：

[QQ FOR LINUX](https://gitee.com/Finch1/QQ)

## 已经实现的功能

- 密码登录
- 群消息
- 断线重连

## 编译&开发

需要以下条件

- dotnet core runtime and sdk  （最新版3.1）
- 手
- VS STUDIO（可选）
- VS CODE （可选）

没有IDE试试下面的命令：

```bash
dotnet publish -c Release
```
