# see-movie

#### 介绍
一款搜索 影视资源的 微信小程序。


#### 软件架构
NODE + Koa2 + cheerio + promise-mysql + WX小程序


#### 使用说明

1. git clone https://gitee.com/crism/movie.git
2. sudo npm install
3. node app


#### 目录结构

- wxcode    小程序
- prototype 原型图
- ./*       后台代码
