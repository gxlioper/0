# 小程序

#### 使用指南

1. 下载本仓库的代码  git pull [地址] master
2. 登录自己的小程序账号，然后去设置里拿到自己的appid
3. 打开 微信开发工具，导入下载的代码，输入自己的appid
4. 在微信开发工具内，找到详情，勾选 `不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书`
5. 然后点击导航条里的  `编译`
6. 接下来就看到小程序运行结果了
