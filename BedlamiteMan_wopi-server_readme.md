# wopi-server
wopi-server是采用java语言开发的Office文件预览服务，该服务需要搭配微软的Office Online Server使用。关于Office Online Server的更多信息详见[微软官网的说明](https://technet.microsoft.com/zh-cn/library/jj219456(v=office.16))，此处不再介绍。
