# Biunovel

这是一个通过配置网站的路径与XPATH自动下载小说的工具。

按章节下载，下载失败的会重试，一直到所有下载成功。

所有下载完成后，会将下载的章节合并，每一部分合并到一个单独的文件中。

## 依赖

- .NET Framework 4.5+

## 读取定义

共享的读取定义存放在 **github** 仓库: https://github.com/hyjiacan/biunovel-getters
