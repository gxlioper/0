# 简介
这是一款基于cocos2dx引擎3.3版本开发的单机斗地主手机游戏。

实现了基本的游戏流程：叫地主/抢地主——出牌/跟牌——出牌提示——分数结算。

实现了不同平台的屏幕适配方案，成功移植到安卓并运行。
