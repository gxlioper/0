#lgh_dev_tools
插件很适用于Maven 管理的多个项目组成工程。主要目的是减少开发人员对svn的重复性操作，
对SVN上的子分支处理，及项目迭代功能上线时，项目之间的版本号依赖处理，及容错检查及处理）
