## 2. 启动前准备
### 2.1 数据库
* 执行sql脚本 doc
* 修改数据库的配置

```text

```

### 2.2 修改hosts
找到C:\Windows\System32\drivers\etc路径下的hosts文件,添加

```text
127.0.0.1  eureka7001.com
127.0.0.1  eureka7002.com
127.0.0.1  config-3344.com
```
