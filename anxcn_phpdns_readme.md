### 环境要求
* CentOS 6/7
* PHP 5.6+(需要支持PDO组件)
* SQLite 3

### 阅读前准备
* 掌握Linux基础知识，熟悉Linux基本命令
* 熟悉网络基础

### 关于DNSmasq
DNSmasq是一个小巧且方便地用于配置DNS和DHCP的工具，适用于小型网络，它提供了DNS功能和可选择的DHCP功能。使用DNSmasq可以很方便的搭建递归DNS（公共DNS），如类似的`119.29.29.29`

### 自建DNS的好处
* 自定义DNS解析
* 屏蔽广告
* 防止DNS劫持

### 适用场景
* 适合公司、家庭等适量用户的小型网络
