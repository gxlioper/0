# Installer-Creator

## Intro

Installer Creator 是一个用来可视化创建 Inno Setup 安装程序的软件。

此软件使用 Qt 开发，支持 Windows 7 及更新的系统。

与其他同类软件相比，此软件具有且不仅有以下突出优点：

- 软件界面简洁易懂，文档详尽，极易上手
- 完全可视化操作，不用写一行代码
- 支持自动生成 Inno Setup 代码
- 丰富的贴图素材及安装程序模板
- 完美支持多国语言
- 多套美观皮肤
- 安装程序高级定制
