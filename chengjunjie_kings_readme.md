# 分布式项目脚手架_kings

#### 介绍
    借鉴当前多方优秀的开源框架，整合现有资源，集成一个最优（对我而言）分布式项目脚手架，目前主要个人使用
#### 软件架构
v1.0 主要借鉴 guns开源项目(https://gitee.com/stylefeng/guns)来进行整理扩展，在此非常感谢stylefeng!
后续文档会定时更新
