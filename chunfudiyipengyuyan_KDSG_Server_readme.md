#KDSG_Server

使用Netty3实现的某卡牌手游服务器，其对应的Unity3d客户端http://git.oschina.net/Spirit_wolf/KDSG_Client

jdk1.6
netty-3.10.6.Final.jar
protobuf-2.2.jar
protobuf-socket-rpc

1 Manager 游戏分区管理+用户注册
2 Deamon  启动游戏分区服务（小）
3 KServer 游戏分区服务器，主函数类为GameServer.java
4 游戏更新服务器（FTP）
5 主数据库
6 从数据库（备份）
7 服务器监控服务（Nagios）
8 统计+找回密码服务（Tomcat）
9 GM
