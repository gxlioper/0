# Excel_JS_Runtime

这是一个能在excel中舒服运行js的代码的框架

#### 运行环境

0. 能执行vba的 office (如:office 2013) 或者 wps (如:wps企业版)

0. 上述两个需要32位版，因为windows系统COM组件的“Microsoft Script Control”只有32位版

#### 开发环境

0. nodejs

0. npm

0. npm 的 http-server

#### 关于release

release/base.xlsm 是发行版了，每次开发的时候就拿这个文件做个副本就可以开始怼了。
