## 特性

- 基于现有的element-ui库进行的二次封装，简化一些繁琐的操作，核心理念为数据驱动视图
- 主要的组件库针对table表格和form表单场景，同时衍生出更多企业常用的组件，达到高复用，容易维护和扩展的框架
- 同时内置了丰富了数据展示组件，让开发变得更加容易

```

$ cd Saber
# 安装
$ yarn install
# 启动
$ yarn run serve     
```
