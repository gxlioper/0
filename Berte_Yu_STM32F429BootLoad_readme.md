# 基于STM32F429IDISC1的硬件基础上的一个BootLoad

----------

## APP地址在 0x08008000处

## 2018-09-13:
	1、上电直接运行BootLoad，通过串口打印信息。
	2、使用Ymodem进行传输bin文件时，接收文件名和文件大小的数据包后就进行FLASH的擦除
	3、FLASH的擦除目前为傻瓜式的从第二扇区开始擦除8个扇区。
	
## 2018-09-13:
	FLASH擦除时，根据传输的文件大小和APP的首地址计算所需擦除的首扇区号和扇区个数，进行按需擦除
