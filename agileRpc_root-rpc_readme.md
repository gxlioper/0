# 分布式 RPC 框架 - 使用说明

基于 http://my.oschina.net/huangyong/blog/361751 博客内容

编译循序:
1) root-rpc install
2) ngbc-rpc-api install
3) ngbc-rpc-service install
4) ngbc-rpc-client install
运行顺序:
1) ngbc-rpc-service RpcBootstrap  -- 开启服务
2) ngbc-rpc-client HelloClient -- 调用服务
