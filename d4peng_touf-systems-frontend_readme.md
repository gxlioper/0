# tofu-systems

## 项目安装相关依赖

```
npm install
```

### 运行开发环境

```
npm run dev
```
`端口默认运行在80端口,确保80端口没有被其他应用占用`

### 项目打包

```
npm run build
```

### 运行 ESLINT 规范

```
npm run lint
```

### 项目结构图

```
├─assets
│  ├─css            页面的CSS样式
│  └─img            页面的图片
├─components        页面的小组件,可以被views使用的
├─icons             图标
├─mock              MockJS
├─router            vue-router
├─store             vue-store
├─utils             vue工具类
└─views             页面级组件
    └─common        通用页面级组件
```
