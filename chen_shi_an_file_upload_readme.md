#### 安装教程

1.  执行doc中的sql文件到指定数据库,修改配置文件中的数据库为自己的,修改https信息,打包执行即可.
2.  把web目录中的文件复制到tomcat 的webapps目录下就行了,然后修改其中的index.html,把里面的url信息改成自己的
3.  直接用了
