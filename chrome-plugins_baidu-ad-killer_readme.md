# baidu-ad-killer

## 介绍

	百度搜索广告终结者，还我干净的搜索结果!

## 使用
1. chrome浏览器打开扩展程序
2. 首先勾选开发者模式
3. 加载已解压的扩展程序(没钱不能上架)
4. 选择本文件夹即可
