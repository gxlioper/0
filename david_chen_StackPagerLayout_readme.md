# StackPagerLayout
StackPagerLayout支持像翻书一样的边沿滑动,继承于AdapterView,实现了控件的复用功能 
 
 

StackPagerLayout.java:
 
    
      reloadAndSmoothMoveNext():重新load下一个view,并平滑过度到该view 
      movePre:前移 
      moveNext:后移 
      smoothMoveToPosition:移动到具体位置 
      Min SDK:Android 4.0
