#ExpressionParser 表达式解析器
简单的用java编写的表达式解析器		

1. 支持+-*/四则运算，
2. 支持>,>=,<,<=,!=,==,&,|,!逻辑运算。
3. 支持三目运算符'?:', 比如 "(1<3)?4+3:5", 输出"7"
3. 支持不带上下文的计算， 输入"(4+2)/(3-1)",输出"6"
4. 支持带上下文的计算， 输入"(a+b+c)==3", 以及上下文{a:1,b:1,c:1}; 输出"true" 
5. 使用方法请参考TestCase.java文件
