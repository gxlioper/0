# BlocklyPi
----
#### 项目介绍
本项目是一个基于Blockly的树莓派可视化编程软件

#### 软件架构
基于HTML，Javascript的在线编辑器


#### 使用说明

见项目 `apps/blocklypi/index.html` 地址

更多说明详见BlocklyPi


#### 实际使用
详见 [http://hgccloud.firadio.net/tools/blocklypi/index.html](http://hgccloud.firadio.net/tools/blocklypi/index.html)
