关于PSI
-------------
>PSI是开源进销存系统。
>
>PSI是希腊字母Ψ的读音。
>
>产品寓意：PSI本身不完美，但追求的是不断改进的品质，距离终极产品就一步之遥(因为在希腊字母表中，Ψ之后是Ω，Ω有终极的意思)。
