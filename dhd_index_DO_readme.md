**【一，框架介绍】** 
DOROODO是一个有组态工程、表单设计、代码生成、图表封装、多项目同时开发、通用业务模块的S2SH开发平台， 
相对传统的信息系统平台，它更侧重于工业监控。 
**【二，框架特点】** 
快：效率提高70% 
好：丰富的组件库，炫丽的界面，稳定的性能 
省：doroodo用的好，可以节省大量的人力资源，熟练的doroodo开发者1个人开发一个中型管理系统不再是难题。
