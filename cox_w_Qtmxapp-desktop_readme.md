# QtGUI 简介
百问网使用的qtgui桌面是基于米尔hmi v2.0版本进行移植适配到imx6ull stm32mp157等开发板上，基于 Qt 5.12 lts版本进行开发，里面包含各种炫酷示如，摄像头图像预览 音乐播放  图片预览 视频播放 设置网络连接 中英文语言切换等等。
