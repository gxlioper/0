# Redis-SpringBoot-Exapple

#### 项目介绍
基于SpringBoot2.0 + spring-boot-starter-data-redis搭建的工程，附带一个多功能工具类，并且 **自定义统一RedisTemplate序列化机制** 

首先安装redis ，此项目对应redis版本为3.x
修改redis为允许远程主机访问，然后启动redis
然后就可以操作了
