# 开源Java 扫雷游戏JMine

#### 项目介绍
开源Java 扫雷游戏JMine，采用Swing界面，是可执行的Jar，在安装好Java运行环境的计算机上，只要双击即可运行。此处提供1.2.5和2.0的可执行jar包和源码下载。

此软件是笔者开源Python扫雷游戏PyMine的祖先。
