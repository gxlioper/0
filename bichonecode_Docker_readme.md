#### 使用说明

1.先创建spingboot的hellodocker小demo
2.使用maven导出jar包
3.使用Xfttp将jar包上传至服务器
4.在服务器上编写Dockerfile文件
5.根据Dockerfile使用jar包创建出hellodocker镜像
6.创建hellodocker的一个容器
7.测试访问
