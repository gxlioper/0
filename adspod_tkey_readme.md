
## Introduction

- **TKey = Token Key** 以 OAuth 2.0 标准为接口设计原则的单点登录系统（SSO）
- **初衷：** 做国内各种登录场景的配件，以完善的学习资料为主核心竞争力
- 纯粹的 HTTP，任意设备、任意场景
- 跨域无状态，随意横向扩展，服务高可用
- Spring Boot 2.1.x 技术栈，没有封装，有一点点基础即刻魔改（MIT License）
- 从开发、测试、部署、监控、前后端分离的材料都已具备
- **最后：没有哪个框架、系统可以套用在任意用户、任意场景、任意需求上，希望对你有思路帮助**
