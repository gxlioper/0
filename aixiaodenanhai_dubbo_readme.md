#dubbo基础DEMO

#mytest01
简单的junit测试模块

#myweb
springMVC调用dubbo远程接口测试模块

#parent
parent 模块没有用到,若其他模块不是在同一个公共项目里面的话,就可以使用parent模块作为 pom.xml 父项约束

#remoteapi
定义远程接口

#service
简单的Dubbo接口模块

#springboot
Dubbo整合springboot进行调用测试

#sso
sso 模块等待后续有时间再完善,（准备使用CAS实现单点登录）
