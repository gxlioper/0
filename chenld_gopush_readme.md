#GoPush
> **GoPush**是开源的分布式推送消息服务.
> * 基于**netty**开发
> * 依赖**redis**
> * 依赖**Java8**
> * 依赖**SpringBoot**
> * 不存在单点问题
> * 拓展开可以做成客服系统
