# 基于Java的chromium浏览器

#### 项目介绍
基于Java的chromium浏览器，使用jar包为JxBrowser 6.23.1

#### 使用说明
开发工具eclipse
环境jdk8或openjdk12
项目编码为UTF-8，启动时需要为jvm附带以UTF-8启动信息
