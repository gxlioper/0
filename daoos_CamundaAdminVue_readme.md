# CamundaAdminVue

#### 介绍
CamundaAdmin前端
基于Camunda模拟器搭建的vue后台管理系统
支持BPMN、CMMN、DMN在线模拟器和在线属性面板编辑，已打通流程模型的数据库存储与调用。

#### 软件架构
软件架构说明
BpmnJS + CmmnJS + DmnJS + Vue Element Admin

#### 安装教程

1.  cnpmn install
2.  cnpm run dev

#### 使用说明
