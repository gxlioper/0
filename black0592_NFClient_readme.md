# NFClient
The clients of NoahGameFrame, which include unity3d client and cocos2d client.


Unity3D:

1) Download the project and open project with unity3d
2) Start NF servers
3) Change the IP as your servers's IP (NFRoot::Start)
4) Run the client
5) Happy fun.
