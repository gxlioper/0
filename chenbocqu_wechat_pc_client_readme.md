# wechat
这是一个基于pyqt的微信PC客户端，通过itchat微信接口登录微信，获取联系人和讨论组，收发消息，更多微信营销功能定制请联系QQ786357601
或加群605395441大家一起来讨论

本产品基于python2.7开发
UI使用的pyqt4
微信接口通过itchat调用web微信接口
