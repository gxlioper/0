#UnityVLC

Unity调用VLC的库解码视频显示到3D场景中，支持常见本地视频和rtmp、rtsp等视频流。

本工程仅为Demo，仅支持Windows平台，做产品请各位商店寻找UMP插件

系统是64位的，因为我用的VLC的类库是64位的，用64位的Unity发布exe时选择x86_64 

示例是播放湖南卫视的rtmp视频流
