# demo-amq

#### 项目介绍
本项目用于演示 `ActiveMQ` 在 `Java` 中使用的 `HelloWorld` 实例

#### 软件架构
本项目为纯 `Java` 应用（非web），使用 `Maven` 进行构建管理。
- jdk：1.7
- activemq：5.14.5

#### 安装教程
> 你的本地必须得装了ActiveMQ才可以测试哟

1. 点击 [这里](https://gitee.com/jastar-wang/demo-amq.git) 进行下载或 `clone`
2. 将本项目导入到 `IDE`工具中
3. 运行 `Consumer` 和 `Producer` 的 `main` 方法即可
