 **Anyfast简介** 

Anyfast是为了顺应快速开发、云端开发的潮流而开发的开源项目。
Anyfast封装了很多简单而实用的类和API，其次又为一些复杂的操作
提供简单方便的API，Anyfast是一套框架，它能提供线程管理、
多线程通信等框架功能。

它实现了很多功能的工具类，并且支持线程高并发下的效率
Anyfast中的Timer计时任务系统在100线程的高并发下，
仍然能实现0误差。
后续我们将添加更多功能！

 **功能** 
- 计时任务调度系统
- 文件操作
- INI文件操作、XML、JSON
- 系统信息（JVM和运行操作系统等信息）
- 一些常用操作（比如MD5、ZIP等）
