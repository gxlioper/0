1. 同时上传论文最终稿的 word 和 PDF 版本，文件标题为：**姓名_学号_论文标题**.
2. 相关资料包括：论文数据，Stata dofile 和 相关程序(若有)。

  - (2) **姓名_Stata** 文件夹设置子文件夹：**Data，Refs，Results**，分别用于存放论文中的数据，参考文献和 dofile 输出的结果。
