# AgeeyeWeb

发现中国[ageeye.cn](https://www.ageeye.cn)网站前端,基于[Quasar Framework](https://quasar.dev/)开发。

## 安装
#### NPM
```
npm install -g @quasar/cli
npm install
```

#### Yarn

```
yarn global add -g @quasar/cli
yarn install
```

## 运行
进入项目目录，执行以下命令：

```
quasar dev
```
