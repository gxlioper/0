# resume-end

#### 介绍
简历系统的后台代码，后台采用python3.7 + Django提供

#### 软件架构
软件采用python3.7 + Django + Mysql,采用前后端分离，JWT  token 机制验证的restfull api   


#### 安装教程

1.  pip install rest_framework
2.  pip install Django
