### 项目说明


jsmile-jfinal是一个基于jfianl框架的权限管理系统。

### 更新说明
1.更改为maven项目
2.升级为jfinal3.1

### 体验地址
http://123.207.187.145:8080/jsmile/      用户名： admin  密码：123456


### 技术选型：

- 核心框架：jfinal
- 视图框架：freemarker
- 数据库连接池：Druid 
- 日志管理：SLF4J 1.7、Log4j
- 页面交互：easyui
- 接口文档管理: Swagger2
- 图标: iconfont

### 软件需求

- JDK1.7+
- MySQL5.5+
- Tomcat7.0+
- Maven3.0+
- nginx
- redis
