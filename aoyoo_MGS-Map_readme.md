#MGS-Map

##概述
Unity3D 制作场景地图，地图轨迹 插件包。

##方案
- 计算小地图UI与场景地形的尺寸比例，获取地图标记的位置。
- 计算地形上标记与世界方向的方向差，获取地图标记的方向。
- 动态生成UI贴图，绘制地图标记运动轨迹。

##实现
- MapFlag.cs 地图标记，例如：漫游角色。
- GlobalMap.cs 全局地图。
- LocalMap.cs 局部地图。
- FlagTrail.cs 地图轨迹。﻿
