# JavaBaas
JavaBaas是基于Java语言开发的移动客户端后台开发框架，其核心设计目标是实现移动客户端的后台结构化数据存储、物理文件存储、消息推送等功能。极大的降低移动客户端的后台开发难度，实现快速开发。

## 主要功能
* 基于文档数据库的结构化数据存储
* 基于ACL的权限管理机制
* 用户系统
* 基于云存储的物理文件存储
* 消息推送
