#beimi 

最新版 0.8.0

开源棋牌游戏，包含麻将、德州、斗地主。首个版本会采用当前最流行的房卡模式。
贝密二开，目前重点修复房卡模式：  
1.目前修复目标为斗地主，以后有时间会逐步修复麻将  
2.修复房间二维码显示错位  
3.修复房卡不能正常加入  
4.修复从房间离开，其他玩家画面仍显示该玩家在房间内  
5.房卡房间加入简单聊天室  
6.当满三人时，有玩家离开，不会弹出二维码房间号，等待其他玩家加入  
7.大家多提bug，有时间我会逐一修复  
8.个人能力有限，改的地方仍有不足之处，请多见谅  

试玩地址：http://47.102.192.61:8888/web-mobile/index.html 加载较慢

贝密游戏是一系列棋牌游戏的名称，其中包含麻将、斗地主、德州，目前正在进行UI设计以及后台系统（JAVA）开发，7月中发布0.1.0版本，仅包含前端UI方案，预计8月份发布首个版本，敬请关注！


1. 开发工具：Cocos Creater
1. 开发语言：Java + JavaScript
1. 服务端框架：Spring Boot + MySQL + JPA + NettySocketIO
1. 客户端语言：Cocos Creater/JavaScript
