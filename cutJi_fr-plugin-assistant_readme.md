## 特性

- [x] 初始化插件

## 快速开始
### 1.命令一键安装
```bash
bash -c "$(curl -fsSL https://gitee.com/cutJi/fr-plugin-assistant/raw/master/install.sh)"
```

### 2.使用
```bash
./frHelper create
```

## 配置说明

支持webroot预配置，添加 `FR_WEB_ROOT`参数即可。例如：
```bash
export FR_WEB_ROOT="/Users/yoqu/Documents/code/self/report-starter-10/webroot"
```


## 支持平台

* Mac
* Linux
