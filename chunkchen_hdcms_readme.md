#安全性
HDCMS完全基于后盾网HDPHP框架开发提供了众多的安全特性，产品安全无忧。这些特性包括：

* COOKIE加密处理
* 数据预处理机制
* XSS安全防护
* 表单自动验证
* 强制数据类型转换
* 输入数据过滤
* 表单令牌验证
* 防SQL注入
* 图像上传检测


#商业友好的开源协议
HDCMS遵循Apache2开源协议发布。Apache Licence是著名的非盈利开源组织Apache采用的协议。该协议和BSD类似，鼓励代码共享和尊重原作者的著作权，同样允许代码修改，再作为开源或商业软件发布。
