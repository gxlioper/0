拷贝的github上的一个项目。
删掉了一些我不用的，留了一个纯粹的推送后台

原项目名称：web-msg-sender
有兴趣的可以去github支持原作者
==============

启动
以debug（调试）方式启动

php start.php start

以daemon（守护进程）方式启动

php start.php start -d

停止
php start.php stop

重启
php start.php restart

平滑重启
php start.php reload

查看状态
php start.php status

查看连接状态（需要Workerman版本>=3.5.0）
php start.php connections
