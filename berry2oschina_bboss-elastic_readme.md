# bboss-elastic

不错的elasticsearch客户端工具包,bboss es开发套件采用类似于mybatis的方式操作elasticsearch

jdk要求： jdk 1.6+

elasticsearch版本要求：2.x,5.x,6.x,+

在项目中导入bboss elasticsearch
maven坐标
```
 
     com.bbossgroups.plugins 
     bboss-elasticsearch-rest 
     5.0.5.0 
 
```
gradle坐标
```
compile "com.bbossgroups.plugins:bboss-elasticsearch-rest:5.0.5.0"
```
