这个项目放的都是些代码示例，欢迎大家一起讨论

- 代码示例有压缩版(rar)、非压缩版。压缩版方便大家下载；非压缩版方便大家查看

- 代码示例使用eclipse编写，若使用其他开发工具(如：myeclipse)，请自行修改所需示例

- 文件编码均是UTF-8

## 介绍 ##
**下面只简单列一下，具体请到相子目录下查看**

- backdevelopment：放着一些后端开发方面的示例
  - basis：数据结构、算法方面的示例
      - algorithm：算法
          - sortingalgorithm：排序算法
          - stringmatchingalgorithm：字符串匹配算法
      - datastructure：数据结构
    - log：日志方面的示例
      - Java：Java日志示例
          - log4j
          - logback
      - paging：分页方面的示例
          - Java：Java分页示例
              - mybatis分页
  - picture：图片操作方面的示例（图片裁剪等）
  - serialize：序列化方面的示例
      - Java：Java序列化示例
          - hessian：使用hession实现序列化
          - java内置序列化：使用jdk自带的序列化实现
          - json：使用json实现序列化
          - protocal buffers：使用protocal buffers实现序列化
          - xml：使用xml实现序列化
  - task：定时任务方面的示例
     - Java：Java定时任务示例
         - Timer实现定时任务
         - ScheduledExecutor实现定时任务
         - (Spring)+Quartz实现定时任务
         - Spring实现定时任务
  - utils：工具包
      - Java：Java工具包
         - com.buaa.utils
              - code：码工具类（二维码、验证码）
              - db：数据库工具类
              - email：email工具类
              - file：文件工具类
              - hadoop：hadoop工具类
              - image：图片工具类
              - protocol：协议工具类（IPV4、IPV6等）
              - remote：远端工具类（Sftp、Scp、Jsch等）
               - safety：安全工具类（RSA、PBE、MD5等）
               - 其他示例，请进入相应子目录查看，这里不再列出
- bigdata：放着一些大数据方面的示例
    - hadoop：放着一些hadoop方面的示例
        - hdfs：放着一些hdfs方面的示例
            - 利用Java API通过路径过滤上传多文件至HDFS
            - 其他示例，请进入相应子目录查看，这里不再列出
        - mapreduce：放着一些mapreduce方面的示例
            - 统计不同工作年限的薪资水平
            - 查找相同字母组成的字谜
            - 统计美国各个气象站30年来的平均气温
            - 处理Excel通话记录
            - 其他示例，请进入相应子目录查看，这里不再列出
- frontdevelopment：放着一些前端方面的示例
  - crypto：加密/解密方面的示例
  - report：报表方面的示例
  - utils：工具包
      - transcoding：转编码工具
          - Unicode转码工具
