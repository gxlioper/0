  + 关闭webpack的严格模式 
    - npm install babel-plugin-transform-remove-strict-mode
    - 在 .babelrc 中加入  { "plugins": ["transform-remove-strict-mode"] }
    - 在启用严格模式的js中 "use strict" 表示使用严格编译
  + 这里ui冲突，现在先不做，以后准备用Vant或者element-ui实现
