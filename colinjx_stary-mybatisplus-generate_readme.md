# stary-mybatisplus-generate

#### 项目介绍
mybatisplus 2.x 代码生成器 3.x版本见stary-mybatisplus3-generate

#### 软件架构
logback、mybatis-plus 2.x、velocity、mysql-connector-java、ojdbc7、sqljdbc4



#### 安装教程

1. 引入依赖
    ` 
         com.stary.mybatisplus 
         stary-mybatisplus-generate 
         1.0.0 
     `
