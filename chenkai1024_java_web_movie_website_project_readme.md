# Java Web电影网站小项目

#### 介绍
基于Servlet+JSP 构建的电影网站，适合初学Java Web

#### 软件架构
   tomcat9 + IDEA


#### 包说明

1. dao 直接操作数据库
2. domain 对应数据库表的实体类
3. exception 自定义异常
4. filter 过滤器包
5. listener 监听器包
6. service 处理各种业务逻辑的包
7. utils 工具包
8. web.servlet.client 客户端的Servlet包
9. web.servlet.management 后台管理的servlet包

#### 使用说明
注意，项目还有些地方存在问题，且项目架构没有清理好，所以存在些用不到的东西，可以忽略。
