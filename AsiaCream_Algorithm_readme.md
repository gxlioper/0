## 树
树：N个节点的集合，当集合非空时，由根节点和子树组成。根节点和子树通过边关联
树叶：某节点下没有左右节点的节点
深度(depth):根到某节点的唯一路径的长。
高度（height）:某节点到树叶的最长路径的长。
根的深度：0
树的高度=根的高度
叶子的高度=0

常见树的遍历
D根
L左
R右
有先序DLR、中序LDR，后序遍历LRD
