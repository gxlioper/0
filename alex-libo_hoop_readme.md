## 它是什么
Hoop是一个基于Spring boot的controller层接口级权限拦截框架 
## 特点
- 基于url的aop拦截
- 集成单点登录
- token验证
- 采用jwt服务端无需存储登录状态
- 提供多种密码加密方式
