# WeChatOrder

#### 介绍
微信点餐

#### 软件架构
springboot2.0.1+maven


#### 安装教程

拉下项目之后用idea导入即可

#### 使用说明

1. 需先下载WeChatOrder-vue
2. 启动OrderApplication.java
3. 运行WeChatOrder-vue
4. 访问localhost:8080可以看到效果
