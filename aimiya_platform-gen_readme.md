# platform-gen

#### 项目介绍
微同软件代码生成工具IDEA插件

#### 使用教程

##### 安装
- 打开IDEA开发工具
- 选择菜单 File -> Settings -> Plugins
    - Install plugin from disk...
    - 选择 platform-gen\.idea\platform-gen.zip 文件，然后点击OK
- Restart
##### 使用
- 重启IDEA开发工具
- 选择菜单 File -> Settings -> Other Settings -> platform-gen
- 添加项目配置，点击OK
- 使用快捷键 ctrl + shift + alt + y
- 填写表名，生成代码
