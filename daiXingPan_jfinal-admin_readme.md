### jfinal-admin 后台框架 
jfinal-admin 是基于JFinal的后台管理系统，采用了简洁强大的JFinal作为web框架，模板引擎用的是beetl，数据库用mysql，前端bootstrap框架。

### 特性
1. 内置用户和权限系统
2. mysql、oracle等多数据库支持
3. 支持引入第三方前端库
4. 基于jfinal_cms深度精简

### 其他
jfinal_cms深度精简，不再依赖jflyfox_base，jflyfox_jfinal。
