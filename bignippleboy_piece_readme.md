Piece.js
=====

Web Framework for Mobile.

## Installation

    $ npm install -g piece

note: you may required using sudo

## Quick Start

  create a project
  
    $ piece create demo && cd demo
    
  create a module
  
    $ piece module member
    
  create a view
  
    $ piece view member login
    
  start web server
  
    $ piece server
