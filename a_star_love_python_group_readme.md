# love_python_group

#### 项目介绍
本项目是纯Python乐学群（779836313）的合作项目，该项目下将创建练习项目，带动群里小伙伴一同学习。

#### 开发环境
- Python 3.6 + 



#### 参与项目

1. 注册一个码云账号，去到https://gitee.com/rogerlee/love_python_group
2. 点击分支，选择正在参与的分支，Fork 该项目
3. 新建 Feat_xxx 分支
4. 提交代码
5. 新建 Pull Request



#### git 工具推荐

- Githubdesktop  在群里文件有安装包
- 点左上角File->Clone repository->URL 填写地址
