# xiaou-open

#### 介绍
小幽编程，免费的开源 - 基础社区项目。
搭建请在网站底部设置‘小幽编程’，链接到小幽官网。
为保证收费版的源码安全，免费版部分源码加密处理。

#### 软件安装说明
https://www.scratchgui.com/open/doc
#### 启动服务注意
必须是node bin/www  不可以是cd bin -> node www
####
mongodb版本：> mongodb-win32-x86_64-2008plus-ssl-4.0.4-signed；
nodeJs版本：> node-v10.13.0-x64
