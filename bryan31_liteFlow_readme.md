## 概述
liteFlow是一个轻量，快速的组件式流程引擎框架，组件编排，帮助解耦业务代码，让每一个业务片段都是一个组件，并支持热加载规则配置，实现即时修改。

## 详细文档请点击：[中文文档](https://bryan31.gitee.io/liteflow)

## 特性
* 提供本地xml的流程配置
* 支持zookeeper流程配置，即时推送修改内容
* 能自由扩展配置源，提供扩展接口
* 支持springboot的自动装配
* 提供串行和并行2种模式
* 消除组件之间参数传递，引入数据总线概念
* 数据槽高并发隔离机制
* 提供无级嵌套条件节点的模式
* 自带简单的监控，能够知道每个组件的运行耗时排行
