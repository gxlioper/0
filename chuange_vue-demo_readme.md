# 饿了么点餐系统

> vue2.0、vuex、vue-router、axios、webpack、eslint、better-scroll

## 安装步骤

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
