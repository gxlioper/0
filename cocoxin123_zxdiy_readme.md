# zxdiy小程序开源版

#### 介绍
zxdiy可作为一个独立的DIY小程序部署集成，支持商城小程序和官网小程序。

#### 软件架构
软件架构说明:微信小程序+云开发

1. 支持自由面板、文本、图片、菜单、搜索、轮播、商品、通告栏组件
