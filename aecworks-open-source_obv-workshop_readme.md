# bimbox-workshop

###运行方式
- 安装依赖包
```bash
npm install 
```

- 运行auth服务
```bash
npm run auth
```

- 运行本地示例
```bash
npm run examples
```
访问localhost:8080可以查看示例
**注意需要将auth/config-production.js中的client_id和secret替换为有效的账号和密码**
