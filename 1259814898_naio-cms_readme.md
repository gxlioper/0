## 微信小程序商城-CMS端

本项目是 [naio](https://gitee.com/1259814898/naio) 微信小程序商城 配套的`CMS管理后台`，基于 [Lin-CMS](https://github.com/TaleLin/lin-cms-vue) 二次开发。

### Deployment

```sh
# clone the project
git clone https://gitee.com/1259814898/naio-cms.git

# install dependency
npm install or yarn

# develop
npm run serve or yarn (run) serve
```
