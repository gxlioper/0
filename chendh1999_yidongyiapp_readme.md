# 移动易——让移动应用开发更容易！

## 简单概述
本项目，主要为了演示移动易APP端与移动易后台交互。

前提是，移动易后台项目已经被启动。具体详见：http://git.oschina.net/sectong/yidongyi

## 核心功能
1. 启动App后校验登录状态，若已登录，直接跳转应用首页；否则，显示登录页面；
2. 支持服务端注册用户API，登录；
3. 后台服务端拉取新闻列表，支持下拉刷新操作。
