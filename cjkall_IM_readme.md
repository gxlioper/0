# IM
一个C/S模式的聊天软件。 
实现功能主要有： 
	服务器端监控； 
	注册帐号； 
	添加、删除好友； 
	文字、表情、截图聊天； 
	查看好友资料； 
	查看历史聊天记录； 
	文件传输。
