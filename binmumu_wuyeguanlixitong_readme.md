#物业管理系统（ssm）
这个软件中用到了  1. 为了分别出  快要过期的  车位 信息，所以，在 每次查询之前，都会有更新车位状态的 存储过程。
                 2. 有备份数据库和 还原数据库。
                3. 收费信息表是 从 水表，电表，燃气表，和价钱表 连接查询然后插入到表中的。
                4. 页面主要通过bootstrap的格栅系统布局的。
