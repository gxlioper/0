#一.实验内容、要求，要求如下：

实现一个用户注册/登录模块，大致功能如下：

1 启动首页为登录页

2 如果用户还没有用户名和密码，则请他/她先进行注册，注册信息写入一个文本文件中，注册成功后直接回到登录页

3 登录成功后则提示XXX登录成功

4 登录不成功则提示究竟是用户名不存在还是密码错误

5 其它你想到的情况，请多参看别的软件的登录页
