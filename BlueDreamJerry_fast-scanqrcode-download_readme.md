### 为什么要写这个
电脑传文件到手机上，百度搜索出来的工具，操作太麻烦，真是烦死了...没找到方便的工具，自己做一个吧。。


### 使用方法
1、命令行模式
```
bin/Fast-ScanQRCode-Download.bat "文件全路径"
```

2、 右键菜单方式（需要添加右键菜单，windows下我写了一个简单的批处理脚本），管理员权限执行install.bat即可
