# AMapPositionPicker

## 概述

[AMapPositionPicker](https://www.oschina.net/p/amappositionpicker)是一款基于高德地图的位置选择插件，使用jQuery开发。

## 特性

主要特性有：

- AMD & CMD 引入
- data-*属性配置
- 初始位置数据
- 浏览器定位
- 字段显示格式、验证
- 数据单向绑定
- 支持地理逆编码
- 外观样式定制
- 工具：在地图上显示点标记
