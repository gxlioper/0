JChart
======

Jingle Chart.移动端专属。极简的图表控件，简单的图形+合适的接口，打造交互丰富的图表。

> 持续开发中...

代码参考了chartjs
地址: http://www.chartjs.org

1. 大幅重构了chartjs的代码
2. 加入了移动端独有的事件(drag\tap\longTap\doubleTap)
3. 通过接口+div的渲染，可以打造出一个华丽的移动端图表
