# GroupControl

#### 项目介绍
GroupControl，Unity群控安卓手机的Demo，目前有截图、点击的功能，其他只有测试(划屏，装软件，输入中文)代码，需要修改

#### 安装教程

1. Unity 2017.4.10f1 (64-bit)
2. Android手机打开开发者选项，打开USB调试

#### 参与贡献

1. Fork 本项目
2. 新建 Feat_xxx 分支
3. 提交代码
4. 新建 Pull Request
