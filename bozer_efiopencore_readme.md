* CPU：Intel Core i7-8750H，3900MHz
* 内存：16 GB 2667 MHz DDR4
* 集成显卡：Intel(R) UHD Graphics 630 (1 GB)
* 独立显卡：GeForce GTX 1060 (6 GB)

* 无线网卡：Intel(R) Wireless-AC 9560
