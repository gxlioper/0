# FruitSlot网络版水果机

*作者历时两个月，独自开发的网络版水果机,客户端+服务器。
  
*client: cocos2d-lua社区版（使用http协议通信协议）
  
*server: go语言  html、jquery（ajax）数据库使用mysql

   1.常规水果机玩法+ -开火车 --小三元 --大三元 --小四喜 --大四喜 --小满贯 --大满贯
