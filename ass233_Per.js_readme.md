#### 项目介绍
Per.js - 快速、简便的响应式JavaScript开发框架。

- 他可以有效的帮助你减少需要编写的代码量
- 他完全是开源可扩展的
- 他的执行时间几乎是Vue.js的8~7分之一倍

#### 浏览器支持

- Chrome >= 1.0
- Firefox >= 4.0
- Safari >= 5.0
- Opera >= 9.5
