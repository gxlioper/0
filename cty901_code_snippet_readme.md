## 介绍
分享代码片段。有access 帮助类，sqlsugar 帮助类等。




#### AccessHelper

#### SqlSugarExtensions

基于sqlsugar orm，自己封装的简易扩展方法类。代码末尾含有使用示例。

#### NPOIHelper

NPOIHelper 帮助类，可以实现DataTable或DataReader导入到excel文件直接下载；或将excel数据导入到DataTable中。
