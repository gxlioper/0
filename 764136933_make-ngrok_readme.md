# Make-Ngrok
One bash to rule ngrok all! Building ngrok server and clients without suffer!

## Dependencies
- Go (1.8)
- Git

## Usage
```
git clone https://github.com/Yaoshicn/make-ngrok.git
cd make-ngrok
./make-ngrok.sh [YOUR DOMAIN] 
```
(Maybe sudo is needed)

## Build Clients
- Linux
    - 32
    - 64
    - Arm
- Windows
    - 32
    - 64
- Mac OS
    - 32
    - 64
