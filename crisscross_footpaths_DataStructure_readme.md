# DataStructure

> 用户课程《数据结构》的教学

#### 使用说明

1. Fork 本仓库到自己的账号下面
2. 在自己的电脑上clone自己账号下面的仓库到本地
3. 由于本仓库的内容可能会时时更新，为了防止与自己仓库的master分支起冲突，建议另新建一个自己名字缩写的分支用于自己的课程练习及代码编写。
4. 按照实验的要求提交内容到自己账号下面仓库的不同分支，分支的名字依次为：
    * 实验1对应的分支名字为：exp01
    * 实验2对应的分支名字为：exp02
    * ......
    * 实验10对应的分支名字为：exp10
    * ......
    * 实验15对应的分支名字为：exp15
5. 具体实验步骤查看实验内容中的每次实验具体内容。
