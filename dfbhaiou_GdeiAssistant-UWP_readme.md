**广东第二师范学院校园助手系统通用Windows平台客户端应用**，最低兼容Windows 10（10.0；版本10240），支持Windows 10桌面版和移动版。应用目前提供了课表查询功能，支持切换深色主题。主要涉及网络和Web服务、JSON数据的解析和序列化、响应式布局设计和基础控件应用、凭据保险箱等技术。应用的后端API数据接口由广东第二师范学院校园助手系统提供。
