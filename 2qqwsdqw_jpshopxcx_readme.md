#### 介绍
系统可以作为常规微商城，也可以作为社区团购系统，开启团长功能，就是社区团购，关闭就是常规微商城 
系统采用Yii2框架+php7.2，后台采用element+vue开发，前后端分离

软件架构说明
采用Yii2框架：一个高性能电商专用框架，用于快速开发现代Web应用程序
