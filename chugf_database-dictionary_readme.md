# database-dictionary

#### 介绍
鉴于企业开发过程中数据字典难维护，我们团队参考阿里云数据库管理中的数据字典展示，进行了一个克隆。
当前支持 MySQL SQLSERVER ORACLE DB2 数据库的数据字典生成，其中PG数据库，正在开发中，后续版本会补起.
