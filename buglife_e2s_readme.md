**e2s(Excel to SQL)**是一款利用JavaFX以及poi技术开发的转换工具，该工具具有以下特点：



- 查看excel中所有sheet的信息
- 自定义sheet名对应的table名
- 自定义sheet中列名对应db表中的字段名
- 提供了自由选择excel文件以及保存sql脚本路径的界面
