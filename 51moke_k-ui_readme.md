# vue-kaiyu-ui

## 安装依赖
```
npm install
```

### 启动项目
```
npm run serve
```

### 编译项目
```
npm run build
```


### 启动文档
```
npm run styleguide
```

### 编译文档
```
npm run styleguide:build
```


### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
#npm 包
```
npm run libs

```
#npm 包发布
```
npm run pub
```
