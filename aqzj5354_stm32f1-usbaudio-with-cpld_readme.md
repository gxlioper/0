# stm32f1-usbaudio-slave

This is the usbaudio 1.0 of stm32 (i2s slave).External CPLD or something must be added to provide exact clock to i2s. 

使用stm32f103系列实现的异步声卡，这个版本增加了CPLD外部时钟源，实现更高精度时钟。
