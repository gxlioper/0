# 牛虎棋牌项目说明
牛虎棋牌是基于ET框架使用Unity引擎开发的手游棋牌游戏。游戏demo: http://planb.swinner.fun:8080
启动游戏需要做一些配置，因为精力有限需要各位亲自己发掘。
## 文件夹说明
```
bt : 腾讯行为树behaivac的项目文件。
design :项目的设计文档。
ET-master:项目源代码文件。
generate :项目的生成工具
FairyGUIProject:Fairygui工程，ui工程

```
